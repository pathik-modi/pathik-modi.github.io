What are the differences between relative, absolute, and fixed positioning?
HTML & CSS are the primary building blocks for making a website. HTML is the contents that get added to the website. Think of all the words, links, images, lists, forms - that's HTML. CSS helps to display that content on the website in a what that makes sense and looks aesthetically pleasing. With that in mind lets dive into CSS positioning.

Positioning on CSS:

Positioning property in CSS specifies the type of position to use for an HTML element (content). this determines how the content is displayed on the website. CSS has 5 positions you can assign to an element: static, relative, fixed, absolute & sticky. Static is the default position assigned to the element. Today we are going to dive deeper to understand the custom position types,

  • Relative
  • Absolute
  • fixed


Relative

  1. Syntax - position: relative;
  2. Relatively positions the element to the place it had when it was static (default position in CSS)
  3. The element saves a blank space in the position it was given under static
  4. Other elements in the section/parent will stay where they were.
  5. By default it stays full width
  6. Its usually assigned relative to the static position. So might not show when you scroll enough on the website


Absolute

  1. Syntax - position: absolute;
  2. It makes the element absolute or fixed but gets assigned to the nearest parent of the element
  3. The element leaves the position it was given when static
  4. Other elements in the section/parent will take its place
  5. By default it changes from full width to auto
  6. Assigned to the top left corner. It gets assigned to the nearest parent so could be just outside the section/parent. It might no not show if you scroll enough on the webpage


Fixed

  1. Syntax - position: fixed;
  2. Bit like absolute but it assigns the element to the viewpoint of a website
  3. The element leaves the position it was given when static
  4. Other elements in the section/parent will take its place
  5. By default it changes from full width to auto
  6. Assigned to the top left corner it will stay fixed there even if you scroll the website
