Arrays and objects

Objects are a non-primitive datatypes. In simple terms it is a container which holds multiple and complex data like a number, string and boolean values (true or false). It can be an array, a function or an object.

Arrays are a list of different values stored together is an array. See below example and syntax of how to make an array and how you can call an array.
let array = ["Pathik", 1, "Pathik One", true, false]
console.log(array[0]) - this will display "Pathik" as arrays are 0 indexed, which means they start counting from 0.

See below example and syntax of how to make an object and howthey can be called
let object = {
name: Pathik,
age: 28,
fast: true,
}
console.log(object.name) - this will display "Pathik"


Functions

Functions are a hug part of JavaScript. They are a type of non-premitive data type or building blocks to hold hold multiple and complex data.

The main difference with objects and functions is that we can call or invoke a function. In very simple terms functions helps us make the code smaller. It is a set of instructions in the form of statements that are stored together in a function. So you can reuse them by invoking or calling them in future.

See below example for a function that multiplies one number to another number.
function multiply (num1, num2) {
alert(num1 * num2);
}

You can call this function anytime you want to multiply two numbers:
multiply(10, 5) - this will directly multiply 10 to 5 and give back 50.

DOM aka Document Object Model

DOM is what holds and binds the HTML, CSS and JavaScript together. It is an object representation of your entire HTML document.

In simple terms DOM is a detailed index page of a very complex book. This index page is holding information of ever single heading, paragraph, image, page number all the way down to every single comma or exclamation point. It has specific names for these data (paragraphs, images etc).

JavaScript uses the indexing style syntax called the DOM to access elements of the HTML document and do all the cool things with them.
